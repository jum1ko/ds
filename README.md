# How Bitter is the Beer?

[Ссылка на соревнование Kaggle](https://www.kaggle.com/competitions/how-bitter-is-the-beer)

---

## Описание Задачи

**Бизнес-проблема:** Пищевая промышленность и пивоваренные компании сталкиваются с вызовом точного прогнозирования горечи пива (IBU - International Bittering Units) на основе его характеристик. Это критически важно для контроля качества, стандартизации вкуса и оптимизации рецептур без необходимости физического тестирования каждого образца.

**Тип задачи:** Регрессия. Модель должна предсказывать непрерывное значение IBU пива по признакам: цвету, содержанию алкоголя, плотности и другим атрибутам.

**Цель:** Построить ML-модель с минимальным RMSE для предсказания горечи пива на тестовом наборе.

|------|----------|-----|
| `id` | Уникальный идентификатор пива | int |
| `name` | Название пива | string |
| `description` | Описание пива | string |

**Ключевой момент:** Целевая переменная - **`ibu`** (только в `beer_train.csv`).

## Метрика Оценки

**Основная метрика:** Root Mean Squared Error (RMSE)

**Формула:**
RMSE = √(1/n * Σ(y_true - y_pred)²)
где \( y_{true} \) - истинное значение IBU, \( y_{pred} \) - предсказанное значение.

**Что это значит:** RMSE измеряет среднюю ошибку предсказания в единицах IBU. Меньше - лучше. Например, RMSE=5 означает, что модель в среднем ошибается на ±5 IBU.
